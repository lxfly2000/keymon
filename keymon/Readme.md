# 按键显示程序
用于东方和 I Wanna 等游戏的按键显示。

* 支持键盘自定义布局。
* 我同时还做了C#版的程序，任选一个使用即可。

## INI文件格式
```ini
[keymon]
width=275 ;窗口宽
height=58 ;窗口高
;字体
fontName=宋体
fontSize=16 ;字号
lineWidth=2 ;线宽
;前景色
foreColorR=255
foreColorG=255
foreColorB=255
;背景色
backColorR=0
backColorG=0
backColorB=0

;<十进制表示虚拟键码>=<左边框>,<上边框>,<边框宽>,<边框高>,<16字以内键名>
160=8,15,66,28,Shift
90=80,15,32,28,Z
88=118,15,32,28,X
67=156,15,32,28,C
37=195,29,24,24,←
38=219,5,24,24,↑
40=219,29,24,24,↓
39=243,29,24,24,→
```
